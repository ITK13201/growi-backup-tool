# GROWI Backup Tool

```shell
GROWI backup tool

Usage:
  growi-backup-tool [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  expand      Expand pages from json to markdown files.
  help        Help about any command

Flags:
  -D, --debug   Debug mode
  -h, --help    help for growi-backup-tool

Use "growi-backup-tool [command] --help" for more information about a command.
```

## Expand

```shell
Expand pages from json to markdown files

Usage:
  growi-backup-tool expand [flags]

Flags:
  -h, --help                    help for expand
  -o, --output-dir string       output directory
  -p, --pages-file string       pages file path
  -r, --revisions-file string   revisions file path

Global Flags:
  -D, --debug   Debug mode
```
